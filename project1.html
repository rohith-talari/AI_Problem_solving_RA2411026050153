📘 Project README

Problem Description

Predicting student academic performance is a classic supervised regression problem. Given measurable study-related features, the model estimates a student's likely exam score on a 0–100 scale.

Input Features

FeatureDescriptionRange
Hours StudiedWeekly study hours before exam1 – 12 hrs
Attendance %Percentage of classes attended40 – 100 %
Previous ScoreScore from most recent prior exam30 – 95 pts
Assignments DoneNumber of assignments submitted (of 10)0 – 10

Algorithm Used

Linear Regression — models the target as a weighted linear combination of input features:

Exam Score = w₁·(Hours) + w₂·(Attendance) + w₃·(Prev Score) + w₄·(Assignments) + b

Weights learned by minimising Mean Squared Error (Ordinary Least Squares):
w = (XᵀX)⁻¹ Xᵀy

ML Pipeline

Raw Data → Handle Missing Values → Train/Test Split → StandardScaler → LR.fit() → Predict → Evaluate

Execution Steps

# Install dependencies
pip install scikit-learn pandas numpy

# Run the desktop app
python student_score_predictor.py

# Steps inside the app:
# 1. Tab ① → Generate Synthetic Dataset (120 rows)
# 2. Tab ② → Click Train Model (20% test split)
# 3. Tab ③ → Enter features → Click Predict Score
